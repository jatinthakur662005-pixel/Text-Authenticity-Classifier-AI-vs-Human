# Text-Authenticity-Classifier-AI-vs-Human


An NLP-based machine learning project that classifies text as AI-generated or human-written using TF-IDF feature extraction and Logistic Regression.

## Overview
With the rise of large language models, distinguishing AI-generated content from human-written text has 
become increasingly important for content moderation, academic integrity, and plagiarism detection. This 
project builds a text classifier that predicts whether a given piece of text was written by a human or 
generated by AI.

## Features
- Text preprocessing and cleaning (lowercasing, noise removal)
- Feature extraction using TF-IDF (unigrams + bigrams)
- Logistic Regression classifier with class balancing to handle imbalanced data
- Model evaluation using accuracy, precision, recall, and F1-score
- Custom text prediction support

## Dataset
The project uses the AuthentiText dataset — a curated collection of labeled AI-generated and human-written 
text samples with additional linguistic features such as perplexity score, burstiness index, and syntactic 
variability.

## Tech Stack
Python, scikit-learn, pandas, NumPy, joblib
